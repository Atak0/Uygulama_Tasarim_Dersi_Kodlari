# Uygulama_Tasarim_Dersi_Kodlari

train_first.py dosyası ilk modeli eğitmek için kullandığım kodları içeren dosyadır.
fine_tune.py dosyası ikinci modeli eğitmek için kullandığım kodları içeren dosyadır.
train_balanced.py dosyası üçüncü modeli eğitmek üzere kullandığım kodları içeren dosyadır.
test.py dosyası modelleri gerçek test verileriyle test etmek için kullandığım kodları içeren dosyadır.
